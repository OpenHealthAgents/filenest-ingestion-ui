# MediVault - Patient Medical Record Intake

A premium, Apple-style frontend for patients to upload, manage, and share medical records.

## Tech Stack
- **Framework**: React (Vite)
- **Styling**: Vanilla CSS (CSS Variables, Glassmorphism)
- **Icons**: Lucide React
- **Routing**: React Router DOM

## Getting Started

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Run the development server:
    ```bash
    npm run dev
    ```

3.  Build for production:
    ```bash
    npm run build
    ```

## Features
- **Dashboard**: Overview of health records.
- **Upload**: Drag-and-drop interface with metadata.
- **Records**: Searchable and filterable list of files.
- **Share**: Manage access and generate share links.
